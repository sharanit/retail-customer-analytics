## How to Rename the Project

### 1. Change Repository Name on GitHub
- Go to repository Settings
- Scroll to "Repository name"
- Enter new name
- Click "Rename"

### 2. Update Local Repository

```bash
# Update remote URL
git remote set-url origin https://github.com/yourusername/NEW-NAME.git

# Verify
git remote -v
```

### 3. Update Project Files

Search and replace the project name in:
- `README.md` - Title and repository links
- `SETUP.md` - Clone command
- `CONTRIBUTING.md` - References
- Any other documentation

### 4. Update README Title

In `README.md`, change the first line:
```markdown
# Your New Project Name
## Statistical Inference with Confidence Intervals and Central Limit Theorem
```
